How to run it
If you have Java installed on your computer, you can run the program using your terminal or command prompt:

Open your terminal in the folder where these files are located/stored on your pc.

Write the following command and press Enter: "java EncryptionDecryptionGUI" (Note: Do not add the .class or .java extension when running this command).
"This will compile the code and create a new .class extension file and you'll have a running program".
